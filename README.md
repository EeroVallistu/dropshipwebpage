# Dropshipping Store Project

This project is a simple dropshipping store built using HTML, CSS, and JavaScript. It includes functionalities for users to browse products, add them to a cart, and place orders. Administrators can log in to add new products to the store.

## Features

- **Product Listing**: View all available products.
- **Add to Cart**: Add products to the shopping cart.
- **View Cart**: See the contents of the shopping cart.
- **Update Cart Count**: Display the number of items in the cart.
- **Place Order**: Checkout and place an order.
- **Admin Login**: Secure login for administrators.
- **Add New Product**: Admins can add new products to the store.
- **Social Media Links**: Links to the store's social media pages.
- **Return to Home Page**: Navigate back to the home page.


## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/EeroVallistu/hobiprojekt.git
    ```
2. Navigate to the project directory:
    ```sh
    cd hobiprojekt
    ```

## Usage

1. Open `index.html` in a web browser to view the store.
2. Use the navigation buttons to browse products, view the cart, and access the admin page.
3. Admins can log in using the username `admin` and password `admin` to add new products.
